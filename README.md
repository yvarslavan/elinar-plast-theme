# Elinar Plast Theme

Custom WordPress theme for Elinar Plast.

## Install
- Copy this folder to `wp-content/themes/elinar-plast`
- Activate the theme in WordPress admin

## Development
- Work from this folder only
- Commit and push using the steps in `GIT_WORKFLOW.txt`

## Notes
- Large video files are kept locally in `assets/video` and are ignored by Git
